# Changelog

## 0.1.0

### Added

- Initial release of the Useless Skills dashboard.
- Terminal table (`--table`) and TOON (`--toon`) output modes.
- Interactive table navigation with sort, filter, and paging shortcuts.
- Per-agent skill usage aggregation from Hermes, OpenCode, Codex, Claude Code, Pi, Continue, Cline, Cursor, Roo Code, and Windsurf.
- On-disk build cache so repeat runs are instant until input sources change.
- `--version` flag and release workflow with version-matched tag validation.
